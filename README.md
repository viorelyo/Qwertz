# Qwertz

## Already Implemented
* Integers, Floats 
* Basic operations : `+-*/()`
* Constants 
* Assignment operator, variables
* Print keyword 
* Strings, strings operations : `+*` (concat)
* Conditional operators : `< <= > >= == !=`
* Branching : `if`, `else`
* Comments : singleline `//` , multiline `/* */` 
* Boolean operators : `&& ||`
* Block statements 
* `for` \ `while` loop


## Features
* Multilanguage support for keywords `RO` \ `EN` \ `DE` \ `RU`
* Code written in separate file
